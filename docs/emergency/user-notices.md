# 유저 공지 템플릿

> **원칙**
> - 법적 분쟁 소지가 있는 단어(도박/불법/사기)는 사용 금지
> - 환불 일정·금액·문의처는 **반드시** 포함
> - 한국어로 작성, 존칭 사용, 경어체

---

## 템플릿 1: 앱스토어 심사 사후 제거

**제목**: [꺼라 중요 공지] 서비스 일시 중단 안내

```
안녕하세요, 꺼라입니다.

저희 서비스가 앱스토어 정책 검토로 인해 {yyyy-mm-dd} 부터 일시
중단됨을 안내드립니다.

■ 환불 안내
- 모든 예치금은 영업일 3~5일 이내 등록된 결제 수단으로 자동 환불됩니다.
- 별도 신청이 필요하지 않습니다.
- 진행 중이던 챌린지는 정상 정산된 후 종료됩니다.

■ 환불 대상
- 예치금 잔액: {balance}원
- 진행중 챌린지 참여금: {activeChallengeAmount}원

■ 문의
- 카카오톡 채널: @kkora
- 이메일: help@kkora.kr
- 운영시간: 평일 10:00 ~ 18:00

불편을 끼쳐드려 죄송합니다. 서비스 재개를 위해 최선을 다하겠습니다.

꺼라 드림
```

## 템플릿 2: 법적 검토로 인한 정책 변경

**제목**: [꺼라 공지] 서비스 정책 조정 안내

```
안녕하세요, 꺼라입니다.

관련 법령에 대한 추가 검토 결과, 다음과 같이 서비스 정책을
조정함을 안내드립니다.

■ 조정 내용
- {변경 사항 간결히}
- 적용 시점: {yyyy-mm-dd}

■ 기존 유저 보호
- 이미 충전하신 예치금은 희망 시 전액 환불 가능합니다.
- 출금 메뉴에서 환불 신청해 주세요.
- 출금 최소 금액(3,000원) 미만도 이번 조치 한정으로 전액 출금 가능합니다.

■ 문의
카카오톡 채널: @kkora

꺼라 드림
```

## 템플릿 3: PG(카카오페이) 계약 변동

**제목**: [꺼라 공지] 결제 수단 점검 안내

```
안녕하세요, 꺼라입니다.

결제 시스템 점검으로 {yyyy-mm-dd}부터 신규 충전이 일시 제한됩니다.

■ 영향 없는 기능
- 잔액 출금 (평소처럼 가능)
- 예치금으로 챌린지 참여 (잔액 내에서 가능)
- 기존 챌린지 정산

■ 제한 기능
- 카카오페이머니 신규 충전

점검 완료 예상 시점: {yyyy-mm-dd}

꺼라 드림
```

## 템플릿 4: 서비스 완전 종료

**제목**: [꺼라 서비스 종료 안내]

```
안녕하세요, 꺼라입니다.

깊은 고민 끝에 {yyyy-mm-dd}부로 서비스를 종료함을 안내드립니다.

■ 일정
- {yyyy-mm-dd}: 신규 가입·충전·챌린지 참여 중단
- {yyyy-mm-dd}: 진행중 챌린지 최종 정산 완료
- {yyyy-mm-dd}: 전 유저 예치금 100% 자동 환불 개시
- {yyyy-mm-dd}: 개인정보 전체 파기 (관련 법령이 요구하는 기록 제외)

■ 환불
- 잔액 {balance}원이 등록된 결제수단으로 영업일 5일 이내 환불됩니다.
- 별도 신청 불필요.

■ 개인정보
- 개인정보보호법에 따라 서비스 종료일로부터 30일 이내 파기됩니다.
- 전자상거래법상 보관 의무가 있는 거래내역(5년)은 별도 보관됩니다.

■ 문의
- 카카오톡 채널: @kkora (종료일 이후 3개월까지 운영)
- 이메일: help@kkora.kr

그동안 꺼라를 사랑해주신 모든 분들께 진심으로 감사드립니다.

꺼라 드림
```

## 템플릿 5: 오픈 연기

**제목**: [꺼라] 정식 오픈 일정 조정 안내

```
안녕하세요, 꺼라입니다.

서비스 정식 오픈 일정을 아래와 같이 조정함을 안내드립니다.

■ 변경 일정
- 당초: {yyyy-mm-dd}
- 변경: {yyyy-mm-dd}

■ 사유
스토어 심사 보완 요청으로 추가 준비 기간이 필요합니다. 더 안정된
서비스를 제공하기 위한 조치이며, 양해 부탁드립니다.

■ 사전 가입자 혜택 (유지)
- 오픈 첫 주 수수료 면제 1회
- 사전 가입자 전용 레벨 뱃지

사전 가입은 그대로 유지되며 재가입이 필요하지 않습니다.

꺼라 드림
```

## 발송 체크리스트

템플릿 발송 전 매번 확인:

- [ ] `{yyyy-mm-dd}`, `{balance}` 등 변수 전부 치환했는가
- [ ] 금칙어(도박/불법/사기) 포함 여부 확인
- [ ] 법무법인 검토 받았는가 (템플릿 1·2·4는 필수)
- [ ] 카톡/이메일/앱/랜딩/X 5개 채널 동시 발송 준비 완료
- [ ] 문의 대응 담당자 대기 중인가
