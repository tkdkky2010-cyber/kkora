# 앱 제거/서비스 중단 대응 플레이북

> **목적**: 애플/구글 심사 사후 제거, 법적 이슈, 카카오페이 계약 해지 등 서비스 긴급 중단 상황에서 **유저 피해 최소화 + 법적 리스크 최소화**.

**트리거 조건 (아래 중 하나라도 발생 시 이 플레이북 가동)**
1. Apple App Store Review Board 앱 제거 통보
2. Google Play 정책 위반 suspend
3. 문체부/금감원/방통위 행정지도 또는 시정명령
4. 카카오페이 가맹점 계약 해지 통보
5. 사행산업통합감독위 조사 개시
6. Firebase/GCP 계정 suspend (약관 위반)
7. 언론 부정 보도로 신규 가입 급감 + 대량 환불 요청

---

## T+0 ~ T+1시간: 즉시 조치

### 1. 서비스 동결 (5분 이내)
- [ ] Cloud Functions `startChallenge` **kill switch 활성화** → 신규 챌린지 참여 차단
  - `functions/src/utils/config.ts` → `SERVICE_ENABLED = false` 배포
  - 또는 Firestore `systemFlags/service.enabled = false` 런타임 토글 (재배포 없이 즉시 반영)
- [ ] `requestDeposit` 차단 → 신규 충전 중지
- [ ] `requestWithdrawal`은 **유지** (유저 환불 출구 보존)
- [ ] 진행 중인 챌린지는 **정상 정산 후 종료** (강제 실패 처리 금지)

### 2. 이해관계자 연락 (30분 이내)
- [ ] 법무법인 긴급 연락 (자문 계약 필수)
- [ ] 카카오페이 담당자 (PG 문의)
- [ ] 공동창업자/주요 주주 (있는 경우)

### 3. 증거 보전 (1시간 이내)
- [ ] 통보 이메일/서한 **원본 PDF 보관** → `docs/legal/incidents/{yyyy-mm-dd}/`
- [ ] 해당 시점 Firestore 전체 export → Cloud Storage
  - `gcloud firestore export gs://kkora-backup/takedown-{yyyy-mm-dd}`
- [ ] 해당 시점 transactions 컬렉션 CSV export (회계 감사 대비)
- [ ] 앱스토어 심사 로그 전체 스크린샷

## T+1시간 ~ T+24시간: 유저 커뮤니케이션

### 1. 공지 템플릿 선택
- 시나리오별 공지문은 [user-notices.md](user-notices.md) 참고
- **금칙어**: "도박", "불법", "사기" — 법적 분쟁 시 자기 불리 진술
- **필수 포함**: 환불 일정, 문의 채널, 책임자 연락처

### 2. 공지 경로 (전부 동시 발송)
- [ ] 카톡 채널 공지
- [ ] 앱 내 풀스크린 배너 (아직 접속 가능한 유저)
- [ ] 이메일 (가입 시 수집한 주소)
- [ ] 랜딩페이지 최상단 공지
- [ ] X·인스타 공식 계정 공지 고정
- [ ] 디스콰이엇 메이커 로그 업데이트

### 3. 환불 안내
- 유저는 아무것도 하지 않아도 됨 (자동 환불)
- 환불 소요 기간: 영업일 3~5일
- 1:1 문의 전담 채널 상시 오픈

## T+24시간 ~ T+7일: 대량 환불 실행

### 1. 사전 검증 (Dry-run)
- [ ] `functions/src/emergency/bulkRefund.ts` 를 `dryRun: true`로 실행
- [ ] 총 환불 대상 수, 총 환불 금액 확인
- [ ] 감사 로그 (`auditLog/` 컬렉션) 생성 확인
- [ ] 잔액 음수/이상치 유저 **별도 분리** (수동 처리)

### 2. 실제 환불 집행
- [ ] 법무법인 서면 확인 후 `dryRun: false`로 실행
- [ ] 배치 500명씩 분할 처리 (Firestore 트랜잭션 한계 고려)
- [ ] PG(카카오페이) API로 실제 이체 or 오픈뱅킹 API 사용
- [ ] 환불 실패 건은 `refundFailures/` 컬렉션에 기록 → 수동 재처리

### 3. 회계 처리
- [ ] 전체 환불 내역 회계법인 전달
- [ ] 세무 조정 (매출 취소 처리)
- [ ] 법인세 영향 확인
