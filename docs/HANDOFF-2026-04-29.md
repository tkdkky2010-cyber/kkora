## 🚨 핵심 결정사항 (4-27 이후 변화)

### 1. 카카오페이 거절 — 한국 PG 길 막힘

**카카오페이 답변 (원문)**:
> "결제금이 환급되는 챌린지는 원천사(은행) 제한 및 채권발생 구조로 내부 규정 상 입점불가 업종에 해당됩니다."

**의미 분석**:
- "원천사(은행) 제한" = 카드사/은행 결제망 자체가 막음 (PG 위 단계)
- "채권발생 구조" = "환급 약속" 자체가 채권 발생, 카드 결제망에서 처리 불가
- "내부 규정 상 입점불가 업종" = 카카오페이 정책상 거절

**파급 효과**: 같은 카드망 사용하는 다른 PG (토스, 네이버 등)도 같은 답변 받을 가능성 매우 높음.

### 2. 챌린저스 모델 자체가 이미 폐기됨

**검증된 사실**:
- 챌린저스는 **2023년 8월 "뷰티득템 앱"으로 리브랜딩**
- 현재 모델: B2B 브랜드 마케팅 플랫폼 (제품 일부 부담 → 도전 → 무료/할인)
- 2024년 매출 150억 (전년 대비 2.5배 성장)
- "예치금 → 환급 → 풀 분배" 모델 자체를 폐기함

**의미**: 우리가 "챌린저스 7년 운영 = 검증된 모델"이라 믿었던 것은 **5년 전 모델**. 챌린저스도 같은 한국 결제 인프라 문제로 피벗.

### 3. 한국에서 본 모델은 구조적으로 불가능

**왜 불가능한가** (3중 벽):

```
[벽 1] 채권 발생 구조
사용자 결제 → "조건부 환급 약속" = 채권
카드사 입장: 가맹점 도산 시 보장 책임 → 거절

[벽 2] 사행성 분류 (풀 분배)
실패자 돈 → 성공자에게 분배 = zero-sum game
카드사 입장: 도박 카테고리 → 거절

[벽 3] 결제 성격 불일치
카드 = "구매" 도구
우리 모델 = "베팅/금융상품"
구조적 불일치
```

**우회책 검토 결과**:
- 무통장입금: 우리가 자금 보관 → 등록 의무 100% 발생 (더 나쁨)
- 토스/네이버/KG/KICC: 같은 카드망 사용, 같은 답변 가능성 높음
- P2P 매칭: 사용자 떼먹기 → 회수 불가 → 작동 안 함
- 신탁 계좌: 자본 + 신탁사 거절 가능
- 카드 Pre-auth: 분배 문제 그대로

**결론**: 한국에서 "본인 돈 + 풀 분배 + 현금 보상" 모델은 사실상 불가능.

## 🌍 글로벌 + 암호화폐 피벗 검토 결과

### 핵심 기술 발견 — Coinbase Onramp + Base 네트워크

**게임체인저**:
- Coinbase Onramp + Base 네트워크 + USDC = **수수료 0%**
- Coinbase가 자기 L2(Base) 보급 위해 USDC 거래 무료 정책
- 한국 PG (1만원에 100원) 보다 더 저렴

### Onramp 수수료 비교 (검증됨)

| 서비스 | 카드 수수료 | 비고 |
|--------|---------|------|
| **Coinbase + Base** ⭐ | **0%** | USDC 한정, 가장 저렴 |
| Alchemy Pay | 0% | USDC 한정, Apple Pay 지원 |
| Stripe USDC | 1.5% | 한국 미지원 |
| Transak | ~1% | 160+ 국가 |
| Ramp | 3.9% (카드) / 1.4% (계좌) | 180+ 국가 |
| MoonPay | 4.5% + spread = 실효 7~8% | 가장 비쌈 |

### 글로벌 풀 설계 — 동아시아 권역 묶음

**사용자 제안 채택**: 3-4시간 차이 권역 묶기

```
[동아시아 권역] (KST ±3시간)
- 한국 KST (UTC+9)
- 일본 JST (UTC+9)
- 홍콩/대만 HKT (UTC+8)
- 싱가포르 SGT (UTC+8)
- 베트남 ICT (UTC+7)
- 인구 합산: 약 16억

KST 22-24시 시작 → KST 5시 종료
ICT 22-24시 시작 → KST 8시 종료
정산: KST 9-10시 (사용자 수령 10-11시)
```

**총 4개 권역**: 동아시아 / 유럽 / 미주 동부 / 미주 서부

### 비즈니스 모델 — 글로벌 크립토 버전

**확정 사항**:
- 참가비: $10 단위
- 수수료: **30%** (회사) / 70% (성공자 분배)
- 한국 시장 포기: 카드 차단 이슈 → 빗썸/업비트 우회는 사용자 책임
- 타겟: 글로벌 영어권 1차

**시뮬레이션** ($10 stake, 1000명, 실패율 20%):
