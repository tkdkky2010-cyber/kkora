# 꺼라 — 앱 심사 거절 대비 문서

## "도박이 아닌 이유" 설명

### 1. 꺼라는 도박이 아닙니다

도박의 법적 정의: "우연의 결과에 의하여 재물을 취득하는 행위"

꺼라는 우연이 아닌 **사용자의 의지와 행동(수면)**에 의해 결과가 결정됩니다.
- 참여자는 자신의 행동(폰을 끄고 자는 것)으로 100% 성공을 통제할 수 있습니다.
- 운이나 확률이 아닌, 자기 절제력이 결과를 결정합니다.
- 모든 참여자가 성공할 수 있으며, 전원 성공 시 모든 참여금이 100% 환급됩니다.

### 2. 유사 서비스 선례

- **캐시워크**: 걸으면 돈을 버는 앱 (건강 + 금전적 동기부여)
- **챌린저스**: 습관 챌린지 앱 (보증금 기반 목표 달성)
- **디지털 디톡스 챌린지**: 스크린 타임 관리 앱

이들 모두 "행동 기반 보증금 반환" 모델로 앱스토어에서 정상 운영 중입니다.

### 3. 보증금 모델의 법적 성격

꺼라의 참여금은 **보증금(deposit)**입니다:
- 참여자가 약속한 행동(7시간 수면)을 이행하면 **전액 환급**
- 미이행 시 보증금이 이행자에게 보상으로 분배
- 이는 부동산 보증금, 호텔 예약 보증금과 동일한 법적 구조

### 4. Apple App Store 가이드라인 대응

**3.1.1 In-App Purchase**: 꺼라의 예치금은 디지털 콘텐츠 구매가 아닌 보증금이므로 IAP 대상이 아닙니다.

**5.3.3 Contests, Sweepstakes, and Raffles**: 꺼라는 추첨이나 복권이 아닙니다. 결과는 참여자 본인의 행동에 의해 결정됩니다.

### 5. 안전장치

- 1일 참여 상한: 10,000원
- 무료 체험 3일 제공
- 카카오페이 1인 1계정
- 미성년자 참여 차단 (카카오페이 본인인증)
- 앱 크래시/배터리 방전 첫 1회 면제

### 6. 법률 자문 예정 사항

- 전자금융거래법 해당 여부 확인
- 통신판매업 신고
- 사행성 리스크 법률 자문
