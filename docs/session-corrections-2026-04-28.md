## 정정 1: 법률 의견서 비용·필요성

### 이전 권고 (2026-04-28 세션)
> "법률 의견서 150~250만원 필수. 안 받으면 카카오페이·앱스토어 통과 어려움"

### 정정 (HANDOFF-2026-04-27 결정 반영)
**5만원 상담 → PG 미팅 → 필요 시 정식 의견서**로 단계 조정.

```
[1단계] 네이버엑스퍼트 5만원 변호사 상담 (이번 주)
        - 핵심 질문 3개 사전 송부 (사업 요약 A4 1장 첨부)
        - 카카오페이 에스크로 위탁 = 전금법 면제 여부 확인
        ↓
[2단계] 카카오페이 비즈니스팀 미팅 (무료, 1~2주 내)
        - sub-account 지원 여부
        - 6개 핵심 질문 송부
        ↓
[3단계] 토스페이먼츠 미팅 (백업, 동시 진행)
        ↓
[4단계] 정식 의견서 (50~200만원) — 필요 시
        - 1~3단계 결과로 합법 명확히 보이면 보류
        - 매출 발생 후 또는 출시 직전 검토
```

### 정정 사유
- 핸드오프에서 **카카오페이머니 네이티브 모델**이 이미 결정됨
- 자금이 우리 통장에 안 들어옴 → 전금법 등록 의무 면제 구조
- 챌린저스 7년 영업의 표준 모델 = 합법성 사실상 검증
- 부트스트랩 단계에서 정식 의견서 100~200만원은 비효율
- **5만원 상담 + 무료 PG 미팅으로 핵심 확인 가능**

### 정식 의견서가 필요한 시점
- 매출 발생 시점 (월 매출 1천만원+)
- 투자 유치 단계 (VC 실사 시 필수)
- 앱스토어 거절 발생 시 방어 자료 필요할 때
- 분쟁(민·형사) 발생 시

## 정정 2: 카카오페이 등록 비용

### 이전 분석 (2026-04-28 세션)
> "정상 30~80만원, 고위험 100~300만원, 권장 500~700만원"
> "보증보험 30~50만원 필수, 고위험 시 50~150만원"

### 정정 (HANDOFF에서 검증된 수치)

**카카오페이머니 네이티브 모델 사용 시**:

| 항목 | 비용 |
|---|---|
| 가맹점 가입비 | **0원** |
| 직결제 수수료 (영세) | **0.56~0.72%** |
| 직결제 수수료 (일반) | **약 1.5%** |
| 토스페이먼츠 PG (백업) | 가입비 22만원 + 연관리비 11만원 |

**중요한 추가 사실**:
- **사용자 본인의 다른 사업체** = 일반 등급 (연매출 30억+)
- **꺼라는 신규 사업자로 별도 등록** → 영세 가능 (사업자등록번호 단위 판정)
- **신규 사업자 첫 6개월**: 일반 등급 자동 적용 → 6개월 후 영세 분류 + 소급 환급
- 매출 적을 때 **자동으로 0.6% 영세 적용**, 매출 많이 나면 1.5% 일반

### 보증보험은 미팅에서 확인 필요
- **카카오페이머니 네이티브** 사용 시: 우리 통장에 자금 없음 → 보증보험 불필요할 가능성
- **토스페이먼츠 PG 사용 시**: 일반적으로 30~50만원/년 (꺼라 고위험 분류 가능성 있음)
- 미팅에서 직접 확인할 항목

### 정정 사유
- 핸드오프에서 검증된 수치가 우리 분석보다 정확
- "카카오페이도 PG임" — 별도 PG 끼울 필요 없을 수 있음 (단독 운영 가능성)
- "결제 ≠ 자금 보관" 구분 필요: 카카오페이로 받아도 우리 통장에 들어오면 등록 의무 발생, 카카오페이 에스크로에 들어가야 면제

- 카카오페이 등록 100~500만원 → **0원 (가입비) + 보증보험 미정** → -100~500만원

**수정된 최소 예산**:
| 항목 | 이전 분석 | 정정 후 |
|---|---|---|
| 사업자등록 | 5~125만 | 5만 (개인사업자) |
| 변호사 상담 | 150~250만 | **5만 (네이버엑스퍼트)** |
| Apple Developer | 14만 | 14만 |
| Google Play | 3.5만 | 3.5만 |
| 카카오페이 가입 | 0~50만 | **0만** |
| 이행보증보험 | 30~50만 | **미정 (미팅 확인)** |
| 도메인 | 2만 | 2만 |

→ **부트스트랩 가능한 최소 예산: 약 130~250만원 수준** (이전 권고 365만원 대비 절반 이하)

## 정정 4: 직결제 수수료 정확한 구간

**중요한 추가 정보 (2026-04-28 사용자 지적)**:
- **0.56~0.72%**: 영세1 구간 (연매출 3억 이하)
- **약 1.5%**: 일반 구간 (연매출 30억+)
- **추후 매출 적을 시 자동으로 0.6%로 내림**
- 신규 사업자는 첫 6개월 일반 등급 → 6개월 후 영세 자동 분류 + 소급 환급

### 의미
- 출시 초기 매출 작을 때는 자동 영세 적용 → **수수료 부담 거의 없음**
- 6개월 후 영세 분류 후 첫 6개월 수수료도 소급 환급
- 매출 30억 넘기면 일반 등급 (그땐 비용 감당 가능)
