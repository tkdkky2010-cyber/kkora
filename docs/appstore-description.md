# 꺼라 — 앱스토어 설명문

## 앱 이름
꺼라 (KKORA)

## 부제
폰을 끄면 돈을 번다

## 설명 (4000자 이내)

밤에 폰을 끄고 7시간 이상 자면 돈을 버는 수면 챌린지 앱, 꺼라입니다.

매일 밤 10시~자정, 참여금을 걸고 챌린지에 참전하세요.
7시간 동안 폰을 켜지 않으면 성공!
실패자의 참여금이 성공자에게 분배됩니다.

### 이런 분들께 추천합니다
- 밤에 폰을 놓지 못하는 분
- 수면 습관을 개선하고 싶은 분
- 작은 동기부여가 필요한 분
- 친구들과 함께 도전하고 싶은 분

### 주요 기능
- 수면 챌린지: 밤 10시~자정에 참전, 7시간 수면 시 성공
- 예치금 시스템: 1,000원 / 5,000원 / 10,000원 참여
- 실시간 전쟁 중계: 생존자 수, 탈락 알림
- 수면 레벨 시스템: 참가자 → 생존자 → VIP → 호스트
- 생존 카드: 성공 시 SNS 공유용 카드 생성
- 무료 3일 체험: 처음 3일은 무료로 체험

### 수익 구조
- 성공: 원금 100% 환급 + 실패자 참여금의 80% 배분
- 실패: 참여금 전액 몰수
- 금요일 밤의 대학살: 수수료 0%!

### 안전장치
- 앱 전환 시 60초 유예 (최대 3회)
- 배터리 방전/크래시 첫 1회 면제
- 카카오페이 안전 결제

## 키워드
수면, 챌린지, 돈벌기, 자기계발, 수면습관, 꺼라, 절제, 스마트폰중독

- 보조: 라이프스타일
